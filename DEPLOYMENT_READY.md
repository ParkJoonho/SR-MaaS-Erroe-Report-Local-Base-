## 배포 환경 설정

### 환경 변수
```bash
NODE_ENV=production
OFFLINE_MODE=true  # 오프라인 환경인 경우
DATABASE_URL=your_database_url
PORT=5000
```

### 시스템 요구사항
- Node.js 18+
- PostgreSQL 13+
- 최소 2GB RAM
- 최소 1GB 디스크 공간
